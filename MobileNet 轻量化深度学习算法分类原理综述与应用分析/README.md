基于 MobileNetV2 的番茄病害识别系统说明文档

一、 项目简介
本项目是基于 MobileNetV2 轻量化神经网络构建的图像分类系统。主要功能是对番茄的健康叶片与晚疫病叶片进行快速识别。系统包含了小样本数据的自动化处理、模型训练，以及一个可视化的检测交互界面。

二、 运行环境
1. 编程语言：Python 3.11 及以上
2. 核心依赖库：tensorflow、numpy、pillow、matplotlib

三、 核心文件说明
1. 数据样本文件夹：存放原始的训练图像数据。
2. prepare_data.py：自动化抽样与数据清洗代码。
3. train.py：构建 MobileNetV2 模型并进行训练的主程序。
4. app.py：带界面的预测系统代码。
5. leaf_model.h5：已经训练好的模型权重文件。
6. training_history.png：模型训练过程的收敛曲线图。

四、 运行步骤
1. 确保所有文件在同一目录下，并且安装了上述依赖库。
2. 如果要查看训练过程：打开终端，输入 python train.py 运行。
3. 如果要直接体验界面检测：打开终端，输入 python app.py 启动软件。在弹出的窗口中点击上传照片即可看到识别结果。
